Paragraph
Paragraph element is used to publish text on the web pages.
<p>This is a paragraph.</p>

The <br> tag is used to insert a line break on the web page.

Creating Horizontal Rules
<hr> tag to create horizontal rules or lines to visually separate content 
sections on a web page. Like <br>, the <hr> tag is also an empty element
example---
<p>This is a paragraph.</p>
<hr>
<p>This is another paragraph.</p>
